### 第四步（附）：Python 3.12 + GPU（必读）

**事实**：在 [JAX CUDA 官方 wheel 列表](https://storage.googleapis.com/jax-releases/jax_cuda_releases.html) 里，**`cp312` + `cuda12` 的「一体式」jaxlib 只发布到 `0.4.29+cuda12.cudnn91`**，**没有** `0.4.35+cuda12` 这类轮子。若强行写 `jaxlib==0.4.35+cuda12…`，pip 会报错；若只装 `jax==0.4.35`，PyPI 上的 jaxlib 往往是 **CPU**，或误装 **`jax-cuda12-plugin`（版本与 jax 不一致）**，就会出现 `sparse_supported`、或 **退回 CPU**。

**再装 `flax` 时不要裸跑** `pip install flax==0.8.4`，否则 pip 可能把 **jax 升到 0.10+**，插件 CUDA 检测失败，仍显示 `CpuDevice`。

**推荐（AutoDL / Python 3.12）**：一键脚本（装 **jax 0.4.29 + 官方 CUDA jaxlib**，并用约束避免 flax 升级 jax）：

```bash
cd /root/autodl-tmp/graduation-thesis
source venv/bin/activate
chmod +x scripts/install_jax_gpu_autodl_cp312.sh
bash scripts/install_jax_gpu_autodl_cp312.sh
```

自检应类似：`devices: [CudaDevice(id=0)]` 或 `GpuDevice`，**不是** `[CpuDevice(id=0)]`。

然后**再** `export LD_LIBRARY_PATH=...`（见本步第一段），再跑消融脚本。

**手工等价命令**（脚本失败时对照）：

```bash
pip uninstall -y jax jaxlib jax-cuda12-plugin jax-cuda11-plugin jax-cuda-plugin 2>/dev/null || true
pip install "jax==0.4.29" "jaxlib==0.4.29+cuda12.cudnn91" \
  -f https://storage.googleapis.com/jax-releases/jax_cuda_releases.html
echo "jax==0.4.29" > /tmp/jax-pin.txt
pip install -c /tmp/jax-pin.txt flax==0.8.4 optax==0.1.9 orbax-checkpoint==0.5.16 chex==0.1.86
rm -f /tmp/jax-pin.txt
python -c "import jax; print(jax.__version__, jax.devices())"
```

与 MuJoCo 无关的提示可忽略。更细的 CUDA11 / 换镜像见 `docs/EDP/AutoDL从零运行指南.md` 第 3.3 节。

### 第四步（附二）：`ptxas` / `sm_90a` / `XlaRuntimeError`

若训练在 `init_rng` / `PRNGKey` 处报错，含 **`ptxas fatal`**、**`.target 'sm_90a'`**，多为容器里 **默认 `ptxas` 太旧**，与 **jaxlib+cuda12** 生成的 PTX 不匹配。处理：

```bash
source venv/bin/activate
pip install 'nvidia-cuda-nvcc-cu12>=12.2'
cd /path/to/graduation-thesis
source scripts/source_autodl_jax_gpu_env.sh
which ptxas && ptxas --version   # 应为 12.x
python -c "import jax; print(jax.devices())"
```

再跑 `run_ablation_multimodal_csi1000.sh`。MuJoCo 相关提示可忽略（ETF/portfolio 不需要）。

## 第六步：汇总结果

```bash
python - <<'PY'
import re
from pathlib import Path
import pandas as pd
latest = sorted(Path("outputs_portfolio_ablation_multimodal").iterdir(), key=lambda p: p.stat().st_mtime)[-1]
rows = []
for p in latest.rglob("progress.csv"):
    m = re.match(r"multimodal_(td3|edp)_s(\d+)_", p.parent.parent.name, re.I)
    if not m: continue
    model, seed = m.group(1).upper(), int(m.group(2))
    df = pd.read_csv(p)
    rows.append({"model": model, "seed": seed, "best": df["best_normalized_return"].max()})
out = pd.DataFrame(rows).sort_values(["seed","model"])
print(out)
print(out.groupby("model")["best"].agg(["mean","std"]))
PY
```
