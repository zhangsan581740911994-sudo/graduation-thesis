## 1) 接下来要做什么？（核心任务清单）

本阶段分为两条线，**目前正在并行推进**：

### 任务 A：生成论文级高清图表（前台优先做，基于已有干净数据）
利用现有的 `progress.csv` 和测试集数据，使用 Python (matplotlib/seaborn) 自动生成以下 8 类图表：

1. **累计净值曲线图 (Cumulative NAV)**：牛市/熊市下 EDP vs Buy&Hold vs EqualWeight。
2. **动态回撤曲线图 (Drawdown)**：直观展示 EDP 在熊市的抗跌能力。
3. **验证集得分曲线 (Eval Return)**：证明离线训练的收敛性。
4. **损失函数下降图 (Loss Curve)**：`qf_loss` 和 `diff_loss` 的收敛过程。
5. **持仓权重热力图 (Weight Heatmap)**：展示模型在不同时间段的调仓逻辑（可解释性）。
6. **每日换手率柱状图 (Turnover Bar)**：证明模型没有被手续费拖垮。
7. **多维度雷达图 (Radar Chart)**：收益、夏普、回撤、波动、胜率的“六边形战士”对比。
8. **多 Seed 收益箱线图 (Boxplot)**：证明算法的稳定性（方差小）。

### 任务 B：深度消融实验（升级版，主线切换到 50 只股票）
在 20 只股票的“干净数据”任务中，EDP 与纯 TD3 表现接近，说明任务复杂度仍不足以拉开差距。为在学术上显著证明 EDP 的优势，后续消融实验需要从“单一 80% 噪声”升级为“高维投资组合 + 多档噪声 + 多 Seed 稳定性”。

- **主线目标（必须达成）**：让 EDP 的 `best_normalized_return` 显著高于 TD3，且跨 Seed 方差更小（稳定性更好）。
- **核心改动 1（优先）**：从 `top20` 升级到 **`top50` 股票池**（更高动作维度、更多相关性结构）。
- **核心改动 2（优先）**：噪声从单一 `0.8` 改为多档 `0.2 / 0.5 / 0.8`，观察难度提升时优势是否扩大。
- **核心改动 3（优先）**：每组至少 `5` 个 Seed（建议 `42,43,44,45,46`），结论以 `mean ± std` 与箱线图呈现。
- **核心改动 4（可选增强）**：加入更强交易摩擦（手续费/滑点/换手惩罚），放大“噪声鲁棒性”差异。

**执行顺序建议**：
1. 先在 `top20` 上做最小升级（多 Seed + 多噪声），检查趋势是否正确。  
2. 立即切到 `top50` 做正式实验（论文主结果以 `top50` 为主）。  
3. 若仍不显著，再叠加交易摩擦与市场分段噪声（牛/震荡/熊不同噪声强度）。  

**论文叙事建议**：诚实保留“20 维简单任务 TD3 已足够强”，并强调“EDP 的价值在高维、多模态、强噪声组合配置中体现得更明显”。

#### 任务 B2（方案 A，推荐用于「做出消融差别」）：多峰 + 低覆盖 行为数据

若「多档 `PORTFOLIO_NOISE_LEVEL` + top50」仍拉不开 TD3/EDP，改用**专门构造的多峰行为策略**离线数据（机制上更利于扩散策略）：

- **实现位置**：`trading_env/portfolio_behavior.py`（`PORTFOLIO_BEHAVIOR_MODE=multimodal`）
- **含义**：每步以概率 `conservative_frac`（默认 0.7）采样**窄 logits**（接近等权）；否则采样 **k 标的尖峰重仓**（多峰、覆盖差）
- **一键脚本**：`bash scripts/run_ablation_multimodal.sh`（默认 `hs300_top50_offline_train/val.csv`，`SEEDS=42 43 44 45 46`）
- **注意**：启动前 **`unset PORTFOLIO_NOISE_LEVEL`**，避免与 multimodal 混用；与 AutoDL 上 JAX 相同，需保证 `LD_LIBRARY_PATH` 含 pip 的 CUDA 库

### 任务 C（补充实验，可选）：用「非沪深300」股票池放大 TD3 vs EDP 差异

**替你选定的方案：中证 1000（CSI 1000）成分股中选取 Top50（推荐按官方权重降序）。**

| 项目 | 说明 |
| --- | --- |
| 为何选中证1000 | 相对沪深300大盘蓝筹，**中小盘占比更高、截面波动与分化通常更大**，更容易暴露「多峰行为 / 价值外推」问题，**有时**比蓝筹池更容易拉开 TD3（偏平滑）与 EDP（生成式多模态）的离线指标差异。 |
| 为何不主观挑「差股票」 | 随意挑 ST、极小盘会被质疑**挑样本、不可复现**；用**官方指数成分 + 固定排序规则**可写进方法章节，答辩更稳。 |
| 与主线的关系 | **仅作附录或扩展实验**，主线仍以当前沪深300子集 + 牛/熊/O2O 为主；正文需写清「不同股票池对应不同风险收益特征，结论不横向替代」。 |
| 备选（二选一即可） | 若权重接口不稳定，可改为：**近 20 个交易日日均成交额最高的 50 只中证1000成分股**（先限定在中证1000内，再加流动性门槛，避免烂流动性）。 |

**实施要点（与现有 top50 流程对齐）**

1. 用 AkShare 等拉取 **中证1000 最新成分及权重**（或成分清单）。  
2. 按上表规则取 50 只，下载与前文一致的日线字段，放入例如 `data_thesis/csi1000_top50_weight_3years_data/`。  
3. 复用 `preprocess_top20_offline_dataset.py` 的逻辑：逐股读入 → 共同交易日对齐 → `return` → 按日期切分 train/val/test，输出例如 `csi1000_top50_offline_train/val/test.csv`（文件名可按你习惯改名）。  
4. 训练/消融脚本中仅替换 `--portfolio_train_csv` / `--portfolio_eval_csv` 路径，**其余超参不变**，便于对比。

**逐步命令与脚本说明**：见 `docs/thesis_materials/CSI1000_Top50实验步骤.md`（含数据下载、预处理、`run_ablation_multimodal_csi1000.sh`）。
